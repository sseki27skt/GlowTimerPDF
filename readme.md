# GlowTimer PDF Presenter

日本語の説明は下にスクロールしてください。

This is a PDF viewer with a built-in timer, designed for lightning talks (LT) and short presentations (e.g., poster sessions). It prioritizes presenter visibility, removes UI controls, and relies on drag-and-drop and keyboard shortcuts for easy, powerful timer management.


## Key Features

* **Drag & Drop**: Load any PDF file just by dropping it onto the browser window.
* **Visual Timer**: The entire window frame acts as a timer, transitioning from `Green` → `Yellow` → `Red` as time runs out.

## How to Use (Controls)

Drag and drop your PDF file onto the window.

| Key | Action |
| :--- | :--- |
| **`Space`** | **Start** / **Pause** timer.<br>*(Countdown only triggers if starting from the full time)* |
| **`→`** / **`Enter`** | **Next Page**. (Resets & stops timer) |
| **`←`** / **`Backspace`** | **Previous Page**. (Resets & stops timer) |
| **`R`** | **Reset** timer on the *current page*. |
| **`F`** | Toggle Fullscreen. |
| **`S`** | Open / Close Settings panel. |

# GlowTimer PDF Presenter （JPN）

これは、ライトニングトーク（LT）やポスター発表のショートプレゼン用に設計された、タイマー内蔵PDFビューアです。発表者の視認性を最優先し、操作UIを排除。ドラッグ＆ドロップとキーボードショートカットだけで、誰でも簡単に高機能なプレゼンタイマーとして利用できます。

## 主な機能

* **ビジュアルタイマー:** 画面の「枠」全体がタイマーとして機能。残り時間に応じて `緑` → `黄` → `赤` へと色が変化し、0秒になると、枠が赤く明滅します。
* **開始カウントダウン:** タイマースタート時に、画面中央に `3, 2, 1` の数字を表示させることができます。

## 使い方

PDFファイルはウィンドウにドラッグ＆ドロップしてください。

| キー | アクション |
| :--- | :--- |
| **`Space`** | タイマーを **スタート** / **一時停止** します。<br> *(残り時間が満タンのときカウントダウンが作動)* |
| **`→`** / **`Enter`** | **次のページ**へ進み、タイマーをリセット |
| **`←`** / **`Backspace`** | **前のページ**へ戻り、タイマーをリセット |
| **`R`** | タイマーをリセット |
| **`F`** | 全画面表示をオン / オフ |
| **`S`** | 設定パネルを開く / 閉じる |
